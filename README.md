

# 🌦️ Weather App

## 🚀 Features

- Search weather by city name  
- Real-time temperature, humidity, and condition  
- 5-day weather forecast  
- Weather icons  
- Dark mode toggle 🌙  
- Loading animation ⏳  
- Responsive glassmorphism UI ✨  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- OpenWeather API
